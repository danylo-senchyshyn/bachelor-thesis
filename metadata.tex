\abstract{%
    % english
   This bachelor's thesis is devoted to the creation of a library for programming the \emph{Robot Karel} in the \emph{C} language. The primitives as well as the sensors of the \emph{robot Karel} were created as standard functions in the \emph{C} language and the \texttt{ncurses} library was used to visualize the situation. The created library became part of the introductory course \emph{Fundamentals of Algorithmization and Programming}, where several thousand students have used it so far solving tasks in exercises and final assignments. Thanks to the library, we are able to solve interesting tasks focused on algorithmization with students from the first lessons, so students can start programming immediately.
}{%
   % slovak
   Táto bakalárska práca sa venuje vytvoreniu knižnice pre programovanie \emph{robota Karla} v jazyku \emph{C}. Jednotlivé primitívy ako aj senzory \emph{robota Karla} boli vytvorené ako štandardné funkcie v jazyku \emph{C} a pre vizualizáciu situácie bola použitá knižnica \texttt{ncurses}. Vytvorená knižnica sa stala súčasťou úvodného kurzu \emph{Základy algoritmizácie a programovania}, kde ju doteraz použilo niekoľko tisíc študentov pri riešení úloh na cvičeniach a záverečných zadaniach. Vďaka knižnici sme schopní so študentmi riešiť zaujímavé úlohy zamerané na algoritmizáciu už od úvodných hodín, takže študenti môžu začať s programovaním okamžite.
}
