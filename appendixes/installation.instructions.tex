\section{Inštalácia zo zdrojových kódov}

To compile this package:

\begin{enumerate}
    \item Normally, you just cd to the directory containing the cmake's \texttt{CMakeLists.txt} file.

    \item Type \texttt{mkdir build \&\& cd build} to create build directory.

    \item type \texttt{cmake ..} to create \texttt{Makefile}

    \item Type \texttt{make} to compile the library

    \item Type \texttt{make install} to install library and the documentation.

    \item You can remove the compiled library and object files from the directory by typing \texttt{make clean}.
\end{enumerate}
